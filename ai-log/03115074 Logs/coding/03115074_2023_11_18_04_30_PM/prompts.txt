    return actors

To extract an actor's name from user input, you can use various natural language processing (NLP) techniques. One common approach involves using named entity recognition (NER) to identify and extract proper nouns, specifically person names, from the input text. Here's an example using Python and the spaCy library for NLP:

First, install spaCy and download the English model:

Now, you can create a function that uses spaCy to extract person names (actors' names) from user input:

python
Copy code
import spacy

# Load the English model
nlp = spacy.load("en_core_web_sm")

def extract_actor_name(text):
    doc = nlp(text)
    actor_names = [entity.text for entity in doc.ents if entity.label_ == "PERSON"]
    return actor_names
You can then use this function to extract actor names from user input:

python
Copy code
user_input = "Can you tell me about Tom Hanks and Julia Roberts?"

actor_names = extract_actor_name(user_input)
if actor_names:
    print("Actor(s) mentioned in the input:", actor_names)
else:
    print("No actor names found in the input.")
This function leverages spaCy's named entity recognition capabilities to identify entities labeled as "PERSON," which often correspond to names of individuals, including actors. Adjustments may be needed based on the specific context and variations in user input.

Remember that NLP for entity extraction is not perfect and might not capture all actor names in complex sentences or in cases with ambiguous references. You might need additional fine-tuning or combine multiple NLP techniques to improve accuracy in extracting actor names from user input.

Certainly! Extracting directors from text involves a similar process to extracting actors' names. You can use NLP techniques to identify proper nouns that might represent director names. Here's an example using NLTK in Python:

python
Copy code
import nltk
from nltk import pos_tag, ne_chunk
from nltk.tokenize import word_tokenize
nltk.download('maxent_ne_chunker')
nltk.download('words')
nltk.download('punkt')

def extract_directors(text):
    tokens = word_tokenize(text)
    tagged_tokens = pos_tag(tokens)
    named_entities = ne_chunk(tagged_tokens)

    directors = []
    for entity in named_entities:
        if isinstance(entity, nltk.tree.Tree) and entity.label() == 'PERSON':
            director_name = " ".join([leaf[0] for leaf in entity])
            directors.append(director_name)

    return directors
This function tokenizes the input text, tags each token with its part of speech, and then uses NLTK's named entity recognition (specifically, the ne_chunk function) to identify named entities, filtering for those labeled as 'PERSON', which might include directors' names.
