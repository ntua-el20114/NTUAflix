To modify the script to exclude specific files like login.py and logout.py, you can add a conditional statement within the loop to skip these files. Here's the revised script:

#!/bin/bash

# Directory containing the .py files
DIRECTORY="/path/to/your/test/files"

# Iterate over each .py file in the directory
for file in "$DIRECTORY"/*.py; do
    # Extract the filename from the path
    filename=$(basename "$file")

    # Skip specific files
    if [[ "$filename" == "login.py" || "$filename" == "logout.py" ]]; then
        echo "Skipping $filename"
        continue
    fi

    echo "Running test: $file"
    python "$file"
    echo "Test finished"
    echo
done
In this script, the basename command extracts the filename from the full path, and the if statement checks if the filename matches login.py or logout.py. If it does, the script will skip that file and continue with the next one. Remember to replace "/path/to/your/test/files" with your actual directory path.
