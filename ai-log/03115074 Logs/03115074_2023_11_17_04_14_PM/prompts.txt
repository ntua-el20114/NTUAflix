import random
import string

def generate_password(length=10):
    characters = string.ascii_letters + string.digits + string.punctuation
    password = ''.join(random.choice(characters) for _ in range(length))
    return password

# Generating a password of length 10
random_password = generate_password()
print("Random password:", random_password)
This script defines a function generate_password() that takes an optional parameter length (default length is set to 10). It uses string.ascii_letters for uppercase and lowercase letters, string.digits for digits, and string.punctuation for special characters. The function then generates a password by randomly choosing characters from this pool for the specified length.

You can call generate_password() without arguments to create a 10-character random password, as shown in the example. Adjust the length parameter in the function call to change the length of the generated password.
